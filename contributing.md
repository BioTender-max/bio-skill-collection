# Contributing

Thank you for considering a contribution to Awesome Bio Agent Skills. This list follows the [Awesome List guidelines](https://github.com/sindresorhus/awesome/blob/main/pull_request_template.md).

## What belongs here

A skill is eligible for inclusion if it:

- Is a self-contained `SKILL.md` folder compatible with Claude-based agent frameworks (OpenClaw, NanoClaw, Biomni, or equivalent).
- Covers a biomedical or bioinformatics topic (genomics, proteomics, single-cell, clinical AI, protein design, etc.).
- Is publicly available in an open-source repository.
- Has a clear, accurate one-sentence description.

## How to add a skill

1. **Fork** this repository and create a branch: `git checkout -b add-my-skill`.
2. **Add your entry** to the appropriate category section in `README.md`, in alphabetical order within the section:
   ```
   - [skill-name](path/to/skill/) - One-sentence description starting with uppercase and ending with a period.
   ```
3. **Check formatting:**
   - Entry name: lowercase-hyphenated, matches the folder name.
   - Description: starts with an uppercase letter, ends with a period, no marketing language, no "awesome".
   - No trailing whitespace.
4. **Open a Pull Request** with the title `Add <skill-name>` and a brief explanation of what the skill does and why it belongs in this list.

## How to add a new category

Open an issue first to discuss whether a new category is warranted. A category needs at least 5 skills to justify its own section.

## Updating an existing entry

If a skill's description is inaccurate or a link is broken, open a PR with the fix. Include a brief note explaining what changed.

## Removing an entry

Skills that are no longer maintained, have broken links, or no longer meet the eligibility criteria may be removed. Open an issue to discuss before submitting a removal PR.

## Code of conduct

Be respectful. Contributions that are disrespectful, spammy, or off-topic will be closed without comment.
