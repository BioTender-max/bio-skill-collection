# SKILL.md - Gregg L. Semenza 蒸馏知识引擎

## 身份
**诺贝尔医学奖蒸馏体** · Gregg L. Semenza (2019) · HIF-1缺氧诱导因子发现者

## 一句话精髓
细胞如何"呼吸"——Semenza发现了HIF-1，揭示了细胞感知和适应氧浓度变化的分子机制，打开了癌症、心血管疾病和贫血治疗的新大门。

## 核心发现

### HIF-1是什么
缺氧诱导因子-1 (Hypoxia-Inducible Factor 1)，一个**氧敏感性异二聚体转录因子**：
- **HIF-1α** — 氧敏感亚基，常氧下降解，缺氧时稳定
- **HIF-1β/ARNT** — 组成型表达，不感知氧

### 氧感知的分子机制
两条氧依赖的**脯氨酸/天冬酰胺羟基化**通路：
1. **PHD羟基化HIF-1α** → VHL识别 → 泛素-蛋白酶体降解（常氧）
2. **FIH-1羟基化HIF-1α** → 阻止p300募集 → 抑制转录活性（常氧）
3. 缺氧时两条通路均抑制 → HIF-1α稳定并激活 → 驱动下游基因表达

### 关键下游靶基因
VEGF(血管生成) · EPO(红细胞生成) · GLUT1(糖代谢) · IGF2(生长) · EMT因子(转移)

## 学术生涯
- Harvard AB → UPenn MD/PhD → Duke儿科住院医 → **Johns Hopkins终生**
- 1995年纯化并克隆HIF-1
- 2001年发现FIH-1

## 科学方法特征
1. **从现象到分子**：EPO缺氧诱导 → 转基因 → 生化纯化 → cDNA克隆
2. **长期深耕**：25+年在同一核心问题上层层深入
3. **方法灵活**：遗传学→生化→细胞生物→生理学自由切换
4. **一生一校**：在Johns Hopkins完成从博士后到诺奖得主的完整旅程

## 医学转化
- **癌症**：HIF-1α在肿瘤中高表达，驱动血管生成、代谢重编程、免疫逃逸、转移
- **靶向药物**：Topotecan(翻译抑制)、PX-12(转录抑制)、Echinomycin(DNA结合抑制)等
- **心血管**：缺血疾病中HIF调控的血管新生
- **贫血**：PHD抑制剂Roxadustat已获批（基于其发现的通路）
