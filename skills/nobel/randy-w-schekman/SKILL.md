## 核心知识蒸馏

### 一句话概括
Schekman用酵母遗传学揭示了细胞内囊泡如何从内质网出芽——发现了COPII包被蛋白复合物，为理解所有真核细胞的蛋白质分泌提供了分子蓝图。

### 核心发现链
```
温度敏感突变体筛选 → 23个sec互补群 → 生化鉴定 → COPII五蛋白复合物
(Sar1 + Sec23/24 + Sec13/31) → 体外脂质体重建 → 机制确立
```

### COPII工作机制
1. **Sar1-GTP** 结合ER膜，启动包被
2. **Sec23/24**（内层）识别货物蛋白（Sec24）+ 激活GTP水解（Sec23）
3. **Sec13/31**（外层）驱动膜曲率和囊泡变形
4. 囊泡出芽，Sar1-GDP释放，COPII解聚

### 与人类疾病
- CLSD（颅锁骨发育不良）→ SEC23A突变
- 先天性糖基化缺陷 → COPII组分突变
- 糖尿病、免疫缺陷 → 囊泡运输功能障碍

### Schekman的独特标识
- 🔬 **遗传学思维**: 三位诺奖得主中唯一以遗传学为核心方法
- 🧬 **酵母信徒**: 坚持简单模型系统的力量
- 📰 **出版改革者**: 抵制CNS奢侈期刊，推动开放获取
- 🧠 **持续探索者**: 诺奖后转向帕金森病和外泌体研究
