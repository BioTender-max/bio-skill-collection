# Sir Peter J. Ratcliffe — 蒸馏知识档案

> 2019年诺贝尔生理学或医学奖得主，细胞氧气感知与适应机制（HIF/VHL/脯氨酰羟化酶通路）的发现者

---

## 身份速写

- **国籍:** 英国 | **领域:** 肾脏病学、细胞生物学、生理学
- **机构:** 牛津大学（Nuffield教授）、弗朗西斯·克里克研究所（临床研究主任）、Ludwig癌症研究所
- **核心标签:** physician-scientist典范 | 冷门领域深耕者 | 谦逊低调的巨匠
- **爵位:** Sir（2014年受封）

## 核心贡献（一句话）

揭示了所有动物细胞如何通过**HIF-1α/脯氨酰羟化酶/VHL**通路**直接以分子精度感知氧气浓度**——氧气是羟化酶的底物，酶活性直接编码氧信息。

## 关键洞察

1. **氧气直接作为酶底物：** 细胞不需要间接传感器，氧气本身就是脯氨酰羟化酶(PHD)的共底物
2. **降解即调控：** HIF-1α持续合成又被持续降解，氧浓度决定降解速率——"制造只为销毁"的优雅Rube Goldberg机制
3. **普遍性：** 该机制存在于几乎所有动物细胞，从线虫到人类高度保守

## 里程碑时间线

| 时间 | 事件 |
|------|------|
| 1989 | 在牛津建立实验室，研究EPO基因的氧依赖调控 |
| ~1995 | 发现HIF-1α的氧依赖性降解，机制存在于所有组织 |
| 1999 | 证明VHL蛋白介导HIF-α的氧依赖降解（Nature） |
| 2001 | 鉴定脯氨酰羟化酶(PHD/EGLN)为真正氧传感器（Science） |
| 2004 | 与Schofield合作综述HIF羟化酶氧感知机制（Nat Rev Mol Cell Biol） |
| 2011 | 获Lasker基础医学研究奖 |
| 2016 | 就任克里克研究所临床研究主任 |
| 2019 | 获诺贝尔生理学或医学奖 |

## 转化影响

- **HIF-PHI药物：** Roxadustat（罗沙司他）、Daprodustat（达普司他）等口服药已获批治疗慢性肾病贫血
- **肿瘤学：** 解释VHL肾癌的血管化机制，推动抗血管生成治疗
- **广泛影响：** 缺氧信号涉及炎症、免疫、代谢、干细胞等上千基因

## 科学方法论特征

- **从临床到基础：** 肾病医生→EPO调控→氧感知分子机制
- **冷门深耕：** 在"不时髦"领域坚持30年
- **学科交叉：** 临床医学×分子生物学×有机化学（与Schofield合作）
- **层层深入：** 基因表达→蛋白质稳定性→蛋白互作→酶学机制→进化保守性

## 哲学观点

- 真正重要的发现不一定在主流方向
