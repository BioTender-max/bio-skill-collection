# Presentation Skills

Tools and workflows for creating presentations and visual documentation.

## Available Skills

### Marp Slides

Build Markdown-based slide decks with Marp. Supports dark themes, mermaid
diagrams, code fragments, and export to HTML/PDF/PPTX.

**Skill file**: [marp_slides.md](./marp_slides.md)
**Template**: [marp_template.md](./marp_template.md)

**When to use**:
- Creating a presentation / slide deck / talk
- Need mermaid diagrams in slides
- Want a single .md file that exports to HTML/PDF/PPTX
